# Data Card

## Dataset Purpose

This repository uses a small offline demo dataset path to exercise the end-to-end pipeline for text review processing. The goal is to demonstrate canonical normalization, annotation, human review, active learning, and training on local demo content.

## Modality

- Text

## Source Types

- Local offline discovery payloads
- Scraped HTML review blocks via demo-friendly local HTML content
- Canonical merged raw records generated by the pipeline

The demo path does not depend on live network sources.

## Canonical Schema

The core canonical schema is:

- `id`
- `source`
- `text`
- `label`
- `rating`
- `created_at`
- `split`
- `meta_json`

After annotation and review, rows may also include:

- `sentiment_label`
- `effect_label`
- `confidence`
- `reviewed_effect_label`
- `review_comment`
- `human_verified`

## Labels

The current pipeline is centered on `effect_label` as the downstream training target (`text -> effect_label`).

Demo label vocabularies are config-driven and may differ between demo themes.

## Human Review Process

Low-confidence rows are exported into a review queue. Corrected labels can be merged back into the reviewed dataset by canonical `id`, and downstream stages continue from the reviewed data.

Label Studio export is supported for annotated batches.

## Known Limitations

- This is an offline demo dataset path, not a large production corpus.
- Demo discovery uses local payloads rather than live data acquisition.
- The data volume is intentionally small.
- The current baseline is designed for reproducible local execution, not for production-scale training.

## Intended Use / Non-Intended Use

### Intended Use

- Local offline demo runs
- Coursework-style pipeline walkthroughs
- Deterministic unit and e2e testing
- Demonstrating the annotation, review, and training flow

### Non-Intended Use

- Production deployment without additional hardening
- Live data collection from external services
- Claims about full dataset representativeness
- Benchmarking as a final model-quality benchmark
